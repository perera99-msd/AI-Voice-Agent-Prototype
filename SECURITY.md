# Security Policy

## Supported Versions
This project is currently a prototype. Only the latest version on the default branch is supported with security fixes.

## Reporting a Vulnerability
If you believe you have found a security vulnerability, please report it privately.

- Email: msdperera99@gmail.com
- Include: a clear description, steps to reproduce, impact assessment, and (if possible) a proof of concept.

## Please Do Not Disclose Publicly
Please do not open public GitHub issues, discussions, or pull requests for security vulnerabilities.

## Response Timeline
We aim to acknowledge reports within 72 hours and will provide an estimated remediation timeline after triage.

## Coordinated Disclosure
We appreciate coordinated vulnerability disclosure. Once a fix is available, we can coordinate a public disclosure timeline if requested.
